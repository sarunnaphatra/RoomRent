ช่วยสร้าง App โปรแกรมจองห้องพัก 
- มีห้อง Mock ระดับถูกไปแพง รวมแล้ว 10 ห้อง
- โดยมีหน้า Login ก่อนเข้าใช้งาน
- username ใช้ admin
- password ใช้ password
- ออกแบบ UXUI ให้สวยงาม
- มีปุ่มเพิ่มห้องพัก (รองงรับเฉพาะที่ Login เข้าระบบด้วย Username : admin เท่านั้น)
- มีปุ่มสามารถ แก้ไข ลบ ห้องพัก (รองงรับเฉพาะที่ Login เข้าระบบด้วย Username : admin เท่านั้น)
- สามารถแก้ไข(เปลี่ยนห้อง เปลี่ยนวันที่เช็คอิน เช็คเอ้าท์ และ ลบ ข้อมูลการจองได้)
- รองรับการใช้งานภาษาไทย และ ภาษาอังกฤษ และ app แสดงภาษาตาม option (ภาษาใน app อ้างอิงตาม option ภาษา TH/EN)
- มีโฟลเดอร์ Backend และ Frontend เพื่อรองรับการใช้งาน TestSprite MCP
- Frontend ให้รันบน Port 4455
- เพิ่มระบบสมัครสมาชิกเพื่อใช้งานสำหรับลูกค้าที่ต้องการจองห้องพักด้วยตัวเอง
- เมื่อลูกค้าสแกนชำระเงินแล้ว ให้แสดงเอกสารการจองขึ้นมา (โดยมีข้อมูล เลขที่การจอง ชื่อลูกค้า , รายละเอียดการจอง ห้องพัก การชำระ , ยอดที่ต้องชำระเพิ่มเมื่อเช็คอิน , สรุปท้ายรายการแบบใบกำกับภาษี)
- ห้องพักไหนที่ถูกจองไปแล้วต้องไม่สามารถกดจองซ้ำได้อีก
- ต้องไม่สามารถเห็นการจองของผู้ใช้งานคนอื่น (ยกเว้น username admin)
- วันที่เช็คเอ้าท์ ต้องไม่สามารถเลือกน้อยกว่าวันที่เช็คอินได้
- เพิ่มการตรวจสอบห้องว่างก่อนจอง (สถานะห้อง ว่าง / ไม่ว่าง ปรับให้อ้างอิงตามวันที่ที่ถูกจองไปแล้วเท่านั้น เช่น วันนี้วันที่ 27/1/2026 แต่ห้องถูกจองไปวัที่ 2/2/2026 เท่ากับว่า ห้องวันนี้ยังว่างอยู่)
- สามารถกดที่ห้องเพื่อเรียกดูวันที่ว่าง (แสดงแบบปฏิทิน) ถ้าวันไหนไม่ว่าง แสดงวันที่เป็นตัวหนังสือสีแดง ถ้าว่างแสดงเป็นสีเขียว
- เพิ่มหน้าจอสำหรับ admin ในการตั้งค่าการเชื่อมต่อ database (รองงรับเฉพาะที่ Login เข้าระบบด้วย Username : admin เท่านั้น)
- สามารถเพิ่มรูปได้มากกว่า 1 รูป
- สามารถแสดงรูปห้องเพิ่มเติมได้เมื่อคลิกห้อง
- ห้อง ปรับเพิ่ม ให้สามารถเพิ่มรูปภาพจากในอุปกรณ์ได้
- วันที่เช็คอิน เช็คเอาท์ ไม่ต้องแสดง T00:00:00.000Z 
- แสดงวันที่แบบ dd-MM-yyyy
- ในหัวข้อการจองของฉัน ช่อง เลขที่การจอง สามารถกดเลขที่การจองเพื่อเรียกดูหลักฐานการโอนเงิน(จากการแนบสลิป)
- ปรับแก้ไข database ให้รองรับการเก็บข้อมูลสลิปโอนเงินของ user / ลูกค้า
- บันทึกข้อมูลการจองลงใน database ที่เชื่อมต่อ (และให้เชื่อมต่อ database อยู่ตลอดเวลาโดยไม่มี timeout)
- หลังขั้นตอนกดปุ่มชำระเงินเรียบร้อย เพิ่มการให้แสดง popup แนบไฟล์สลิปหลักฐานการโอนก่อนจะแสดงหน้าสรุป พร้อมปรับปรุง database ให้รองรับด้วย
- สามารถเปลี่ยนรูปภาพ QR Code ตอนชำระเงินได้ ควรมีเมนูตั้งค่า
- เพิ่มการตั้งค่า การเชื่อมต่อ ส่งไลน์ / telegram เมื่อลุกค้ากดเสร็จส่งหลักฐานการจอง (ส่งข้อมูลหน้าใบเสร็จรับเงิน + รูปภาพสลิปการโอน)
- บันทึกการจองไปที่ google calendar (googleCalendarId = "s2.sarunnaphatra@gmail.com)
- หากมีการยกเลิกการจอง ให้ลบข้อมูลวันที่จองใน google calendar ออกด้วย
- ปรับแก้ไข icon ใหม่ทั้งหมด ให้ดูทันสมัย และ สวยงาม
- หากห้องไหน กดเลือกวันเช็คอินแล้วพบว่าซ้ำซ้อน (มีการจองไปแล้ว) ให้ popup หน้าต่างแสดงว่า วันที่ลูกค้าเลือกถูกจองแล้ว
- เปลี่ยนปุ่ม จอง ที่เป็นไม่ว่าง ให้สามารถจองได้ปกติ แต่จะเป็นการตรวจสอบว่าวันที่จองนั้น ห้องว่างหรือไม่ว่างแทน
- ในการจอง ข้อมูลลูกค้าเพิ่ม field กรอกข้อมูลเบอร์โทรศัพท์ลูกค้าด้วย พร้อมปรับ database ให้รองรับด้วย
